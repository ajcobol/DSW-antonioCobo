Son tecnicas usadas por css3 que permiten que los estilos de tu página web se ajusten a los distintos dispositivos y a los distintos tamaños de pantalla y orientaciones (vertical u horizontal). Entre las recomendaciones de diseño de sitios mas actuales se recomienda primero diseñar para dispositivos móviles.

Imagen 1: Layout escritorio. propiedad de w3schools

Imagen 2: Layout movil. w3schools

Los media query deben cumplir algunas condiciones para poder ser usados. A continuación un ejemplo de como usar los media query

@media only screen and (max-width: 1024px) {
  body {
    background-color: lightblue;
  }
}

Algunas distribuciones populares por dispositivos:

//Dispositivos muy pequeños (telefonos con menos de 600 pixeles)
@media only screen and (max-width: 600px) {...}

// Dispositivos muy pequeñoss (tablets y teléfonos grandes, 600px y superiores)
@media only screen and (min-width: 600px) {...}

// Pantallas medianas (tablets en horizontal, 768px y superiores) */
@media only screen and (min-width: 768px) {...}

// Pantallas grandes (Monitores y portátiles con mas de 992px)
@media only screen and (min-width: 992px) {...}

// Pantallas muy grandes (Pantallas con más de 1200px de ancho)
@media only screen and (min-width: 1200px) {...}

Entre las distintas pantallas o dispositivos podremos variar cualquier elemento CSS que deseemos, por ejemplo el tamaño de la fuente, su color, ancho de columnas o la forma de mostrar el menú ó simplemente ocultar elementos visuales.
